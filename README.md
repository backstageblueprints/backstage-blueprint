# Backstage Blueprint — Platform

The web platform for Backstage Blueprint: the marketing landing site, the course, and
(in a later phase) the Toolkit.

Deployed via Netlify continuous deployment — every push to `main` rebuilds and redeploys.

See `PLATFORM_BLUEPRINT.md` and the phase runbooks in the parent project folder for the
full architecture and build plan.

## Status

Phase 1 — in progress (landing page, paid course, gated viewer).
